# JavaScript Calculator App

A clean and lightweight calculator built using **HTML, CSS and Vanilla JavaScript**.

This project demonstrates real-world DOM manipulation, keyboard interaction, and UI rendering without any frameworks.

---

## Features

- Button-based calculator interface
- Full keyboard support (numbers and operators)
- Live calculation using JavaScript
- Error handling for invalid expressions
- Calculation history display
- Premium dark theme UI

---

## Usage

1. Clone the repository
2. Open `index.html` in any modern browser
3. Start calculating using mouse or keyboard

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Learning Scope

This project covers:

- Event listeners (keyboard & click)
- DOM manipulation
- JavaScript expression evaluation
- UI design and layout
- Error handling
